## Bali-Inspired Coastal Bungalow – Concept & Planning

This repository presents the concept development of a **modern tropical, Bali-inspired bungalow** intended for coastal living in Kenya.

The project explores a **biophilic design approach**, blending architecture with nature through open spaces, integrated greenery, and seamless indoor-outdoor transitions. The home features **three ensuite bedrooms**, a **courtyard swimming pool**, and a layout designed for natural ventilation and passive cooling.

It serves as a personal portfolio piece documenting the **design vision, inspiration, and phased construction strategy**.
